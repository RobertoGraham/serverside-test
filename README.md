# serverside-test

## System Requirements

* Java 8
* Maven 3.6.1

## Running the app

Run the following commands in the root directory of the project:

```text
mvn install
mvn exec:java -Dexec.mainClass="robertograham.serversidetest.Main"
```

## Running the tests

Run the following command in the root directory of the project:

```text
mvn test
```
